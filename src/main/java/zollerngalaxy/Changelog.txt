===== CHANGELOG =====
Zollern Galaxy v2.0.39
=====================

[NEW]
* Added a config option to enable Mineshafts to spawn on Kronos.
    - Set to false by default.
* Added a config option to disable Sweet Foods giving Potion Effects.
* Added Azurite Ore gen to Kronos.
    - These ores will only spawn in Obsidian veins, and do not explode.
* Added fruit tree generation to Oasis.
* Added more biomes to Oasis.
* Added an API for modders to add "custom" Star Gates.
    - Two dimension IDs can be set, and the Gate will work the same way as the rest.
* Added a new Star Gate to travel between the Overworld and the Moon, using the new Custom Star Gate API.
* Added Small Crater spawns to Eden.
* Added crop generation to Eden.
* Added Batte Tower spawns to Eden.
* Added a GC Boss Dungeon to Eden.
* Added the Crawler boss mob.
* Added the Kree mob.
    - Drops Kree Flesh on death, with a chance to drop their Reaper Axe.
* Added the planet Maveth.
    - The Kree spawn here.
    - Inspired by Agents of S.H.I.E.L.D.
* Added the Sentinel mob.
    - Inspired by No Man's Sky.
    - These will spawn on most (not all) planets and moons from this mod.
        - This can be disabled in the config.
* Added compatibility with Aliens vs Predator mod.
    - Several entities from that mod will now spawn on Centotl if it is present.
    - This can be disabled in the config.
* 

[CHANGES]
* Radite Ingots and Blocks can now be used inside of a Magma Crucible to make Radite Fluid.
* Registered fluids to the Ore Dictionary.
* Oil lakes will now generate on Oasis unless disabled in the config.
* Reworked Oasis biome decoration.
* Oasis' "Red Sea" ocean biome has been completely reworked and is now no longer completely flat under the water.
    - This is not perfect, but it is a step in the right direction.
* Increased amount of oxygen per tank tier.
* Oxygen Tanks from this mod are now craftable.
* ZG Village farms will now generate with Obsidian instead of Logs.
* Any item in the mod that gives off fuel (e.g. Super Charged Coal) will now show its burn rate in its item description.
* Updated a few planet and star textures.
* Reworked Eden chunk generation and biome decoration.
* Modified crater gen config option for Tlaloc to be for all planets.
* Changed Xenomorph Queen's boss health bar color to purple.
* Reworked Crystal Towers entirely.
* Created placeholders for all planned future planets.
* Increased health and attack damage of Xenomorphs.
* Facehuggers and Xenomorphs will now spawn naturally on planet Centotl.
* Vexbots will now spawn on Xantheon.
* Centotl now has a breathable atmosphere.
* 

[FIXES]
* Fixed Tlaloc Dungeon Chest having the wrong render texture (thank you MJRLegends).
* Fixed Bowl of Popcorn food item requiring regular Corn Kernels instead of Popcorn.
* Fixed a non-translating tooltip.
* Fixed tunnel gens not generating correctly.
* Fixed Crystal Towers not generating correctly.
